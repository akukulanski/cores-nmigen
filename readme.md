
# nMigen cores

Miscelaneous cores described in nMigen and tested with cocotb.

## Tools

Requirements:

* nMigen
* Yosys v0.9+
* Cocotb
* Icarus Verilog v10.1 (reported problems with v10.2+)
* nmigen-cocotb

Getting started: See provided dockerfile `.gci/Dockerfile`
